# 人體偵測運動遊戲
此專案可以讓使用者知道他目前做的運動動作是否標準，當前可偵測深蹲、仰臥起坐、開合跳、抬腿四個動作
### 實作方式
使用四肢的座標點去判斷使用者是否有達到該運動預期的幅度
